2. Displaying Your First Text

Note: The HUD will automatically update when the profile file is saved. Meaning that you can go back into game to see the changes everytime you hit save!

Text written in profiles are displayed verbatim on the hud. There are three exceptions:

  • {stuff} ( like {fps} )
  • &character ( like &7 or &b )
  • Lines that have only ==stuff== and =stuff=

Instructions

We're going to be starting from scratch. You can either pick a profile that is blank, or delete the contents of one. Let's go ahead write something!
