7. Theming

Theming allows you to customize the style of your profiles, like foreground and background color, font, scale, text shadow, and line spacing

Reference: All the Theme Options

Instructions

Let's change the background color and make the text bigger, and make one of the lines use a different background color for the last line.

  • Remove the TopRight section. Why? Because the scale I use below is comically large
  • At the top of the profile
    • Add ==BackgroundColor:#88880000==
    • Add ==Scale:1.5==
  • Add =BackgroundColor:#88008800= before the last line on the TopLeft section
    • This one only works on v3.0.0 and above

Profile File

profile1.txt
==BackgroundColor:#88880000==
==Scale:1.5==

==Section:TopLeft==
fps: &a{fps}
xyz: &c{bx} &a{by} &b{bz}
=BackgroundColor:#88008800=
biome: &e{biome}

Result In-Game
