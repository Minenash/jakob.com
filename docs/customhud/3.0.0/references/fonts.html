Built-In Fonts

Profile Theme: ==Font: <font_identifier>==
Local Theme: =Font:<font_identifier>=

Minecraft has a few built-in fonts that I've slightly modified so that characters not in the font still works

Font Identifier Alphabet
Default minecraft:default
Illager (Alphanumberic) minecraft:illageralt
Illager (Alphabet Only) minecraft:illageralt2
Standard Galatic Alphabet
(Enchanting Table)		 minecraft:alt
