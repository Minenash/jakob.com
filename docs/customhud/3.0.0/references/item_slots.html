Item Slots

Most Item Variables have a <slot> argument. The table below shows the available slots you can pick from.

Slot Shorthand
weapon.mainhand mainhand, main, weapon
weapon.offhand offhand, off
armor.head head
armor.chest chest
armor.legs legs
armor.feet feet
hotbar.0 - hotbar.8 h0 - h8
inventory.0 - inventory.26 i0 - i26
