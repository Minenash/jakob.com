Reference: Bukkit Color Codes

Custom Colors

Use this if you want a specific color and you have the hex code for it. Just don't forget the first two digits is the opaqueness of the text.

Syntax Example
&{AARRGGBB} &{77FF4040} Coral Red (50% transparency)
&{RRGGBB} &{FF4040} Coral Red (no transparency)
&{ARGB} &{BF50} Orange-ish Red (75% Opacity)
&{RGB} &{F50} Orange-ish Red (no transparency)
Letters Meaning Range
AA or A Alpha (Translucency) 00-FF or 0-F
RR or R Red 00-FF or 0-F
GG or G Green 00-FF or 0-F
BB or B Blue 00-FF or 0-F
